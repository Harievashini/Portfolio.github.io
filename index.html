    Graduate projects

    Dessert Suggestion, created a web application to suggest nearby dessert shops and their famous desserts along with nutrition content. The application also displays recipe information for the given dessert. It is developed using HTML, CSS, Python, flask and bootstrap with GCP’s Datastore as the backend database. Used GCP’s container registry for building the docker image and cloud run to deploy the application. Places API, Recipe API, Nutritionix API are the different APIs used to develop this application.

    To Do App Development and Monitoring, created and deployed a web application using Postgres database backend, integrated with Gunicorn Application Server on GCP. Monitored the traffic to the website using Kibana, Elastic search and maintained the reliability of the site.

    Prediction of Wine quality using Random Forest and Decision Tree, choses to explore Decision tree and Random forest algorithms to predict the wine quality as perceived by a wine drinker on the Wine dataset. The project aims at training the models and experimentally determining the effects of various parameters and comparing the results of both the algorithms.

    Solving the game of 2048 with Artificial Intelligence, is based on Q learning and the design involves the use of neural networks as the function approximation method. In 2048 there are 4 random components, that is, initial configuration of the game, addition of random tiles after very move, exploration of the agent and unavailability of moves. This approach is based on prioritized experience replay where the model trains to learn best game-playing strategy from the experiences collected.

A custom scheduler that successfully schedules pods on nodes with maximum memory available was developed and tested it against the default one.

    MongoDB – NoSQL database, used Freeway dataset to work on MongoDB. Designed a data model, inserted the Freeway data into collections and Executed queries using Pymongo.

    Comparison of String Searching Algorithms, the motivation behind this project was to discuss various String Searching algorithms,compare and analyze their execution time and space complexity on different length of patterns, etc.

    Internet Relay Chat Application, describes a simple Internet Relay Chat (IRC) protocol by which clients can communicate with each other. The system employs a central server which 'relays' messages that are sent to it to other connected users. Users can join rooms, Users can also send private messages directly to other users.
